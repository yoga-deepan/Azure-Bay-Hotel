# Azure Bay Hotel - Luxury Room Booking Website

A premium animated hotel booking website built with React, Tailwind CSS, and Framer Motion.

## Features

- Fully responsive design (mobile, tablet, desktop)
- Smooth animations with Framer Motion
- Glassmorphism UI design
- Auto-sliding testimonials carousel
- Interactive booking form with success popup
- Smooth scroll navigation
- Premium luxury hotel aesthetic

## Tech Stack

- React JS (Vite)
- Tailwind CSS
- Framer Motion
- React Icons
- Google Fonts (Playfair Display + DM Sans)

## Installation & Setup

### 1. Install Dependencies

```bash
npm install
```

### 2. Run Development Server

```bash
npm run dev
```

The website will be available at `http://localhost:5173`

### 3. Build for Production

```bash
npm run build
```

### 4. Preview Production Build

```bash
npm run preview
```

## Sections

1. Navbar - Sticky navigation with glassmorphism
2. Hero - Full-screen with animated background
3. About - Hotel information with stats
4. Rooms & Pricing - 4 room types with pricing
5. Amenities - Premium hotel facilities
6. Testimonials - Auto-sliding customer reviews
7. Booking Form - Interactive reservation form
8. Footer - Contact information and links

## Customization

- Colors: Edit `tailwind.config.js` for color scheme
- Fonts: Modify Google Fonts link in `index.html`
- Images: Replace Unsplash URLs in components
- Content: Update text in respective component files
